## What is Express.js?
Express is a lightweight Node.js framework that makes
backend development easier and cleaner.

## Why Express?
- Simple routing
- Cleaner code
- Middleware support
- Industry standard

## Routing
Routes define how the server responds to requests.

Example:
app.get("/", (req, res) => {})

## Response Types
- res.send()
- res.json()
- res.status()
